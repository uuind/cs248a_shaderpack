# CS248A Winter 2026 Final Project --> Minecraft Shader

## Shader Features
We focused primarily on translucent material effects, which includes more realistic water rendering and foliage movement.
* Water Effects
    - Surface Reflection
    - Depth Attenuation
    - Perturbed Normals (waves)

* Foliage Effects
    - Waving crops and leaves

* Celestial Light
    - More realistic day/night lighting
